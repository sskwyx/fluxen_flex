## 🧩 О проекте

**Flex** --- это open-source платформа, позволяющая владельцам и
разработчикам Minecraft-серверов запускать собственный сайт, личный
кабинет и систему донатов **без EasyDonate и аналогичных сервисов**.

Проект создан как полностью контролируемая альтернатива SaaS-донат
платформам.

> ⚡ Ваш сервер --- ваши правила --- ваша монетизация.

## ✨ Возможности

### 🎮 Для серверов

-   💳 Собственная система донатов
-   👤 Личный кабинет игроков
-   📦 Управление привилегиями
-   💰 Баланс пользователей
-   📜 История покупок
-   🔐 Авторизация пользователей

### 🧑‍💻 Для разработчиков

-   Open Source архитектура
-   Полная кастомизация UI
-   i18n мультиязычность
-   Next.js + React + TypeScript
-   Готовая база под собственный проект

## 🚀 Быстрый старт

### Предварительные требования

-   Node.js 18+ → https://nodejs.org
-   npm

Проверка установки:

``` bash
node -v
npm -v
```

------------------------------------------------------------------------

## 📦 Установка

``` bash
npm install
```

------------------------------------------------------------------------

## 🧪 Запуск разработки

``` bash
npm run dev
```

Откройте:

    http://localhost:3000

------------------------------------------------------------------------

## 🏗 Production сборка

``` bash
npm run build
npm start
```

------------------------------------------------------------------------

## 🚀 Deploy

Проект поддерживает:

-   ✅ Vercel
-   ✅ VPS / Dedicated Server
-   ✅ Docker
-   ✅ Node.js hosting

### Deploy через Vercel

1.  Загрузите репозиторий на GitHub
2.  Перейдите на https://vercel.com
3.  Import Project
4.  Deploy

------------------------------------------------------------------------

## 📁 Структура проекта

    app/            — страницы Next.js
    components/     — UI компоненты
    lib/            — логика и i18n
    public/         — изображения
    styles/         — глобальные стили
